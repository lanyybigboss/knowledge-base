# Agent Skills 技能体系

## 概念定义

**Skill** 是一种轻量级、开放式的格式，用于通过专业知识和工作流扩展 AI Agent 的能力。

**类比**：如果把 Agent 比作员工，Skill 就是员工掌握的能力。

## 常见技能类型

| 技能类型 | 示例 |
|---------|------|
| **信息获取** | WebSearch、文档检索、数据库查询 |
| **文件操作** | xlsx（Excel）、docx（Word）、pdf（PDF处理） |
| **工具调用** | 浏览器自动化、API调用、命令执行 |
| **内容生成** | 文本生成、图像生成、代码生成 |
| **数据处理** | CSV处理、JSON解析、数据转换 |

## 本项目可用技能

### 知识库管理相关
- **文件解析**：PDF、DOCX、图片OCR
- **AI分析**：Ollama + DeepSeek 双通道
- **搜索**：Fuse.js 模糊检索
- **同步**：跨模式数据同步（Vite/Electron）

### 办公自动化
- **xlsx**：Excel 文件读写
- **docx**：Word 文档处理
- **pdf**：PDF 文档处理
- **browser**：浏览器自动化

## 使用原则

1. **按需调用**：根据任务类型自动选择合适技能
2. **技能组合**：复杂任务可组合多个技能
3. **失败降级**：技能不可用时应有备选方案
4. **结果验证**：对技能返回结果进行合理性检查

## 技能调用流程

```
用户请求 → 分析任务类型 → 选择合适技能 → 执行技能 → 结果处理 → 返回给用户
```
